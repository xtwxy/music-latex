    呵呵，用\verb+lilypond-book+ 整的文档可以混合文档和音乐。
    例如，干这事：

    \begin{lilypond}
      \paper {
  #(define fonts
    (make-pango-font-tree "Noto Serif CJK SC"
                          "Noto Sans CJK SC"
                          "Noto Sans Mono CJK SC"
                          1))
}
        \relative c' {
          c2 g'2 \times 2/3 { f8 e d } c'2 g4
        }
        \addlyrics {
            Hello!你好！
        }
    \end{lilypond}

    选项用方括号括起。例如下面是一个片段，需要用片段选项：
    \begin[fragment,quote,staffsize=26,verbatim]{lilypond}
      c'4 f16
    \end{lilypond}

    大体积的音乐、文档也可以用
    \verb+\lilypondfile+来把它放到分开的文件中。

    \begin{lilypond}
        \score {
            \relative c' {
                c2 g'2 \times 2/3 { f8 e d } c'2 g4
            }
        }
        \markup {
            die, die, die, die, 死!
        }
    \end{lilypond}

    \begin{lilypond}
        melody = \relative c' {
            \time 4/4
            c4 d e f
        }
        verse = \lyricmode {
            你 好 世 界
        }
        \score {
            <<
            \new Voice = "soprano" { \melody }
            \new Lyrics \lyricsto "soprano" { \verse }
            >>
        }
    \end{lilypond}
